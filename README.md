# 読書管理アプリ

これは授業で作成した、読んだ本を記録・管理するためのWebアプリケーションです。

## 主な機能
- Google Books APIによる書籍情報の自動取得
- 読了／未読のステータス管理
- 感想メモの記録・編集
- 読みたい本リストの管理
